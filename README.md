# 🚀 DisciplineAI – Smart Productivity & Habit Tracker

DisciplineAI is an AI-powered productivity and habit tracking platform designed to help users build discipline, stay focused, and improve daily performance.

It provides a modern dashboard to manage tasks, track habits, monitor focus time, and analyze productivity — all in one place.

---

## ✨ Features

- ✅ Task Management (Add, track, complete tasks)
- ✅ Habit Tracking (Daily consistency tracking)
- ✅ Focus Timer (Deep work sessions)
- ✅ Productivity Dashboard (Analytics & insights)
- ✅ Google Authentication (Secure login)
- ✅ Real-time Activity Tracking
- ✅ Scalable backend with FastAPI

---

## 🛠 Tech Stack

- **Backend:** FastAPI (Python)
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite / PostgreSQL
- **Authentication:** Google OAuth
- **Other:** Redis, Celery (for background tasks)
